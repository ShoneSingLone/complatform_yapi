# workspace
- [ ] 打开 workspace，整体是资源管理器的风格：左侧树（Tree），右侧详情（List/Grid + 详情面板）。右侧支持宫格/列表切换、排序、分页/无限滚动（按产品定位二选一），并且各类型都支持搜索（搜索字段随资源类型变化）。
  - [ ] 左侧 Tree
    - [ ] 节点类型：个人空间、Group、Project（可选：按需展开到文档/接口等二级节点）
    - [ ] 交互：单击选中并刷新右侧；双击/展开按钮展开子节点；右键菜单（重命名/移动/删除/复制链接/在新标签打开，按权限显示）
    - [ ] 状态：加载中骨架屏、无权限提示、空节点提示；支持记忆上次展开/选中（localStorage）
  - [ ] 右侧详情区
    - [ ] 顶部：面包屑（个人空间 / Group / Project）、搜索框、视图切换（宫格/列表）、排序（最近更新/创建时间/名称）、筛选（我创建的/我关注的/成员/状态/标签）
    - [ ] 列表模式：名称、类型、描述、Owner、更新时间、权限/可见性；支持多选（批量关注/移动/删除）
    - [ ] 宫格模式：卡片展示关键字段 + 快捷操作（打开/关注/更多）
    - [ ] 详情侧栏（可选）：选中项的简介、成员、标签、最近活动、快捷入口
  - [ ] 默认个人空间
    - [ ] 右侧默认展示：私人项目（我创建/我参与）+ Follow 的项目（含分组或 Tab）
    - [ ] 快捷创建：新建项目/导入项目（按权限显示）
  - [ ] Group 切换
    - [ ] 左侧 Tree 除个人空间外为各 Group；点击 Group 后，右侧切换到该 Group 概览页（并更新面包屑、URL query 以便分享/刷新保持）

## Group
- [ ] 点击 Group，右侧是 Group 概览
  - [ ] 基础信息：名称、描述、可见性（公开/内部/私有）、Owner、创建时间
  - [ ] 统计/概览：项目数、文档数、成员数、最近活跃（近 7/30 天）
  - [ ] 最近活动：最近更新的项目/文档，支持一键打开
- [ ] Group 的 Projects
  - [ ] 列表/卡片：名称、描述、Owner、更新时间、成员数、状态（正常/归档）
  - [ ] 操作：创建项目、归档/取消归档、转移 Owner、删除（按权限）
  - [ ] 搜索/筛选：名称/描述、Owner、标签、状态
- [ ] Group 的 文档
  - [ ] 入口：文档列表（可按项目聚合）/文档库（按标签/目录）
  - [ ] 字段：标题、所属项目、作者、更新时间、可见性、标签
  - [ ] 操作：新建文档、移动到项目/目录、分享链接、权限设置（按权限）
- [ ] Group 的 Members
  - [ ] 角色：Owner/Admin/Member/Guest（可按产品定义）
  - [ ] 邀请：按邮箱/用户名邀请，支持批量；邀请链接（可选）
  - [ ] 管理：改角色、移除、禁用；成员搜索与筛选（角色/加入时间）
- [ ] Group 的 Settings
  - [ ] 基础设置：名称/描述/头像、可见性、默认权限策略
  - [ ] 安全与权限：成员加入策略、项目创建权限、外链分享开关
  - [ ] 高级：删除 Group（强校验）、审计日志（可选）


## Project
- [ ] 点击 Project，右侧是 Project 概览
  - [ ] 基础信息：名称、描述、所属 Group、Owner、可见性、标签
  - [ ] 统计/概览：API 数、文档数、成员数、最近更新
  - [ ] 快捷入口：创建 API/创建文档、导入 Swagger/OpenAPI（可选）
- [ ] Project 的 APIs
  - [ ] 列表：名称、路径、方法、状态（草稿/发布/废弃）、负责人、更新时间
  - [ ] 搜索/筛选：路径/名称、方法、状态、标签、负责人
  - [ ] 操作：新建/编辑/复制、发布/回滚、生成客户端/Mock（可选）、权限控制
- [ ] Project 的 文档
  - [ ] 结构：文档目录树 + 右侧列表/详情（可选与 workspace 复用布局）
  - [ ] 功能：新建/编辑/版本历史（可选）、评论（可选）、权限（可选）、导出 PDF/Markdown（可选）
  - [ ] 搜索：标题/正文（可选全文）、标签、作者、更新时间
- [ ] Project 的 Members
  - [ ] 角色：Owner/Maintainer/Developer/Viewer（可按产品定义）
  - [ ] 管理：邀请、改角色、移除；支持从 Group 继承（可选）与独立成员（覆盖策略明确）
- [ ] Project 的 Settings
  - [ ] 基础设置：名称/描述/头像、可见性、标签、归档
  - [ ] 集成：Webhook、OpenAPI 导入导出、访问 Token（可选）
  - [ ] 风险操作：转移项目、删除项目（强校验 + 二次确认）
