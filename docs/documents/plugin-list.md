## 怎么分享我的插件？

Fork [xspace](https://github.com/YMFE/xspace), 然后修改 docs/documents/plugin-list.md, 修改完成后请 Pull-Request.

## 插件列表

-   [dingding](https://github.com/zgs225/xspace-plugin-dding) 钉钉机器人推送插件
-   [qsso](https://github.com/ymfe/xspace-plugin-qsso) sso 第三方登录
-   [import-rap](https://github.com/wxxcarl/xspace-plugin-import-rap) 从 rap 中导入项目
-   [export-docx-data](https://github.com/inceptiongt/xspace-plugin-export-docx-data) 数据导出 docx 文档
-   [import-swagger-customize](https://github.com/follow-my-heart/xspace-plugin-import-swagger-customize) 导入指定
    swagger 接口
-   [autotest](https://github.com/duicym/xspace-plugin-autotest) 定时自动测试发送钉钉插件
