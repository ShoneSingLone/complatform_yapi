## iconfont 图标抓取方法

用本地 Chrome + agent-browser 从 iconfont 抓取 SVG：

1. 启动 Chrome 调试：`chrome.exe --remote-debugging-port=9222`
2. 连接：`agent-browser --cdp http://localhost:9222 open "URL"`
3. 等待：`agent-browser --cdp http://localhost:9222 wait 3000`
4. 获取 SVG：`agent-browser --cdp http://localhost:9222 eval "document.querySelector('.page-search-container svg')?.outerHTML"`

iconfont 搜索页直接用 `.page-search-container svg` 即可获取第一个搜索结果 SVG。

## XSpace Desktop 业务图标

- 图标目录：`static/business_xspace/assets/svg/`
- 命名规范：`_{业务名}.svg`（下划线前缀）
- 文件名用下划线格式：`setting_outlined.svg` 而非 `SettingOutlined.svg`
- 代码中引用：`icon="_settings"` 对应 `settings.svg`

- `Dock.vue`：layout-grid → `_layout_grid`，settings → `_settings`
